# AI-Agent-MCN-Solution
## 🔗 飞书自动化接入说明（基于 Coze + 飞书官方插件）

本项目采用 **Coze（扣子）无代码工作流** 串联大模型生成与飞书文档存档，实现"输入 Brief → 自动产出脚本 → 自动落库到飞书云文档"的端到端闭环。以下是完整复现步骤：

### 步骤 1 · 在 Coze 创建 Bot

1. 登录 Coze 平台，进入「个人空间」→「+ 创建 Bot」
2. Bot 名称填写：`小红书脚本助手 · 轻醒项目`
3. Bot 类型选择「**工作流模式**」
4. 简介填写："输入品牌 Brief 与对标博主资料，自动生成小红书种草脚本并归档至飞书"

### 步骤 2 · 配置大模型节点 + 注入 System Prompt

1. 在工作流编辑器中拖入「**大模型节点（LLM）**」
2. 在「人设与回复逻辑」字段中，**完整粘贴** `prompts/system_prompt.txt` 的内容
3. 在「输入参数」中声明 4 个变量：
   - `brand_brief`（String，多行）
   - `target_blogger`（String，多行）
   - `scene`（String，单行）
   - `duration`（String，默认值 `60-90s`）
4. 在「输出格式」中选择 `Markdown` 格式输出。

### 步骤 3 · 飞书自动化写入配置

1. 在 Coze 插件中心搜索并添加官方插件：`飞书` 
2. 授权登录并选择 `创建文档 / 写入文档` 操作节点。
3. 将上一步大模型节点的输出（`final_script`）作为变量，映射并传递给飞书节点的“文档内容”参数中。
4. 运行工作流，即可实现脚本的自动化归档。
